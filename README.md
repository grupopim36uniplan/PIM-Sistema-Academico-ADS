Sistema Acadêmico Colaborativo com IA

Este repositório contém o projeto desenvolvido no PIM II do curso de Análise e Desenvolvimento de Sistemas (ADS).

O objetivo é criar um sistema acadêmico colaborativo que una praticidade, inclusão e tecnologia, com o apoio de um chatbot de Inteligência Artificial para auxiliar os usuários.

Estrutura do repositório

- codigo-c → contém o código de cadastro e armazenamento de alunos, desenvolvido em C.
- codigo-python → scripts em Python responsáveis pela leitura dos dados e interface gráfica (Tkinter).
- diagramas → diagramas UML e esquemas do sistema.
- prototipos → telas desenvolvidas no Figma ou Canva.
